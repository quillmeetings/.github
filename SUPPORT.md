# Support

## Using Quill

If you need help with the Quill app — questions, account issues, or something
not working as expected:

- 📖 Help center: [quillmeetings.com/help](https://quillmeetings.com/help)
- ✉️ Support: [support@quillmeetings.com](mailto:support@quillmeetings.com)

GitHub issues are **not** monitored for product support requests — please use
the channels above so we can help you properly.

## Contributing to a project

- **Bug reports and feature requests** — open an issue in the relevant
  repository using its issue templates.
- **Questions about contributing** — see
  [CONTRIBUTING.md](https://github.com/quillmeetings/.github/blob/main/CONTRIBUTING.md).

## Security

Security issues follow a separate, private process — see the
[security policy](https://github.com/quillmeetings/.github/blob/main/SECURITY.md).
