# Code of Conduct

The Quill organization has adopted the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
(version 2.1) as the code of conduct for all of its repositories and
communities.

We expect everyone who participates — contributors, maintainers, and users — to
help us keep these spaces welcoming and respectful.

## Reporting

To report a concern, contact
[conduct@quillmeetings.com](mailto:conduct@quillmeetings.com). All reports are
reviewed and handled confidentially.

The full text of the code of conduct is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
